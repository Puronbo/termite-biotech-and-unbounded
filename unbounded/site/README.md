# UNBOUNDED — Site

The web front-end of the UNBOUNDED ecosystem. Contains four files:

- **`portal.html`** — The main entry point; a stylized launch page with navigation to all project areas.
- **`dashboard.html`** — A data dashboard for browsing biofuel research results and experiment logs.
- **`index.html`** — A secondary landing/index page.
- **`favicon.svg`** — The UNBOUNDED logo in SVG format.

Open **`portal.html`** to start your journey.
