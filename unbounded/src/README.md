# UNBOUNDED — Source

Contains **`gateway.py`**, the unified CLI gateway for the Termite Project and UNBOUNDED Vision.

- Offers a 10-menu system: browse research data, open web pages, read documents, run simulations, and more.
- Launch with `python gateway.py` from this directory, or `python ../run.py` from the project root.
- Integrates with both the `unbounded` and `termite-projects` directory trees.
