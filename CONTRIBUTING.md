# Contributing to Termite Biotech + UNBOUNDED

Thank you for your interest in contributing! This repository is a community-owned project built by whoever shows up.

## Where to Start

This is a monorepo with three sibling projects. Please see the relevant contributing guide for the project you want to help with:

- **UNBOUNDED platform** — [`unbounded/CONTRIBUTING.md`](./unbounded/CONTRIBUTING.md)
- **Termite Research Initiative** — [`termite-projects/README.md`](./termite-projects/README.md)
- **ChronoSend** — [`chronosend/CONTRIBUTING.md`](./chronosend/CONTRIBUTING.md)

## Quick Guidelines

1. **Read the Code of Conduct** — we are building a respectful, harassment-free community.
2. **Explore the issues** — look for `help wanted` and `good first issue` labels.
3. **Open a discussion** — not sure where to start? Open a discussion, not an issue.
4. **Fork and build** — pick something and start.
5. **Open a pull request** — see our template in `.github/PULL_REQUEST_TEMPLATE.md`.
